자바스크립트의 데이터 입/출력

        * window : 자바스크립트 내장 객체. 브라우저 창이 열릴 때마다 하나씩 만들어지는 객체
                    브라우저 창 안에 존재하는 모든 요소들의 최상위 객체(생략 가능)

        * document : 웹 문서마다 하나씩 만들어지는 객체(HTML 문서에 대한 정보를 가짐)

1. 데이터를 출력하는 구문

- [window.]alter("알림창에 출력할 문구");
- [window.]console.log("콘솔창에 출력할 문구");
- document.write("화면상에 출력할 문구");
- 선택한 요소.innerHTML | 선택한 요소.innerText = "요소에 출력할 문구";
