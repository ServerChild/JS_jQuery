jQuery 개요

* jQuery란?

- 기존의 자바스크립트 언어만으로 구현하기엔 복잡했던 구문 작성들을 간소화함
- '존 레식'에 의해 만들어짐
- javascript 기반 라이브러리
즉, javascript언어로 유용한 함수나 내용들이 이미 정의되어있는 모음

=> 복잡한 자바스크립트 구문을 보다 짧고 쉽게 개발 가능

- jQuery의 장점
> DOM요소와 관련된 스크립트를 보다 쉽게 구현할 수 있음
> ajax(비동기식 통신), 이벤트 처리 등 폭넓게 지원함
> jQuery와 관련된 확장형 플러그인, 오픈 api등을 지원함(챠트, 슬라이드, 등등)

* jQuery 라이브러리 연결 방법

head 태그내에 script태그로 외부 jQuery 관련된 파일 연결
오프라인, 온라인 방식이 있음
- jQuery 공식 사이트 : 주소
- 방법 1. jQuery를 직접 다운로드 받아 경로 지정해서 사용
- 방법 2. CDN 방식

* jQuery 구문 작성하기

jQuery 구문으로 DOM요소를 다루는 구문을 기본적으로 작성하기 때문에 문서상에 요소들이 다 만들어지고난 후 실행되어야 함

* 순수 자바스크립트 
        window.onload = function() {
            실행 내용;
        }

        javascript에서의 window.onload 구문은
        해당 문서의 요소들을 모두 로드하고, 연동된 외부 문서들도 가져온 후 실행함
        단, 한번만 작성 가능
jQuery에서는 3가지 구문으로 제공 
        [1] jQuery(document).ready(function() {
                실행 내용;
            })
        
        
        [2] $(document).ready(function() {
                실행 내용;
            })

        
        [3] $(function() {
                실행 내용;
            })

        해당 문서의 요소들만 로드되면 바로 실행됨
        여러번 기술 가능(제한없음)하며, 순서대로 실행
